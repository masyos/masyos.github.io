### IDE

#### Delphi スタイル (1 window 化) にしたい

IDE がデフォルトでは古い Delphi スタイル (複数 window) なので、新しいスタイルで作業したい場合.

1. package install
    1. AnchorDocking 1.0 
    1. AnchorDockingDsgn 1.0
1. rebuild IDE
1. package install
    1. sparta_DokedFormEditor 0.0
1. rebuild IDE
